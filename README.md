## Featured Project

### certAIn — AI-Powered Project Planning & Risk Forecasting Platform

certAIn is my capstone project and the strongest example of how I combine **AI, project thinking, data workflows, and project-management logic** into one end-to-end platform.

It helps move project planning from static timelines toward **AI-assisted, dependency-aware, probabilistic forecasting**.

#### What it does
- converts natural-language project briefs into structured project plans
- validates outputs with schema-based logic
- models dependencies as a DAG
- calculates critical paths
- runs Monte Carlo simulation for delivery forecasting
- provides **P50**, **P80**, delay probability, and risk-driver insights
- supports scenario comparison through a stakeholder-facing Streamlit interface

#### Stack
Python, Streamlit, SQLite, Pydantic, NetworkX, Pandas, NumPy, Plotly, LangChain, ChatGroq, Docker, GitHub Actions
